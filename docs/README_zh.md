# RelayGo

**一个 Go 语言实现的个人 API 转发服务，旨在统一管理与未来扩展多平台 API 访问。**

---

[English](../README.md)

## 🌟 特性

- **统一 API 请求转发**：集中管理和转发来自不同源的 API 请求。
- **简化访问管理**：旨在简化对多源 API 的访问和管理。
- **多平台扩展能力**：架构设计支持未来轻松扩展到更多平台。
- **Go 语言构建**：利用 Go 语言的并发和性能优势。
